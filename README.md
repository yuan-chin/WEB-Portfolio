# Yuan Carlo M. Chin — OJT Portfolio Website

This project is my personal portfolio website created in preparation for my On-the-Job Training (OJT). It serves as a professional platform where I present my projects, technical skills, certifications, and background as a BSIT student specializing in Web Development.

The purpose of this website is to clearly demonstrate my capabilities in frontend and full-stack development while maintaining a clean, structured, and user-focused design.

# Project Overview

The portfolio highlights my selected works, including real-world and academic projects. It provides detailed descriptions of my contributions, the technologies I used, and the problems each project aimed to solve.

The design follows a minimal and professional style, focusing on clarity, readability, and smooth user experience across different devices.

# Key Features

Responsive and mobile-first layout

Dark and light mode toggle

Interactive project detail modals

Skills and certifications section

Resume preview and download

Contact section for internship opportunities

# Technologies Used

HTML5

CSS3

JavaScript

Google Fonts

# Purpose for OJT

This portfolio was developed to support my OJT application by:

Showcasing my technical skills and growth as a developer

Presenting completed projects with clear explanations

Demonstrating my understanding of responsive design and structured development

Providing an accessible way for companies to review my work
